# HiMIRT
Regularized Multidimensional Item Response Theory Modeling  Examples

[HiMIRT_Functions.R](https://github.com/DogeinSky/HiMIRT/blob/main/HiMIRT_Functions.R) contains all functions for the examples.

[HiMIRT_Example.R](https://github.com/DogeinSky/HiMIRT/blob/main/HiMIRT_Example.R) contains all examples for *lasso HiMIRT* and *card HiMIRT*. One can run the code line by line to see how the proposed methods perform.
